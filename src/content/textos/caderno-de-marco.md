Nem todo projeto precisa nascer com ambicao de escala. Alguns so precisam durar o suficiente para formar linguagem propria.

## 1. Fazer pequeno ainda e fazer serio

Existe um engano recorrente em tratar projetos compactos como ensaios descartaveis. Mas o tamanho nao determina profundidade. Muitas vezes, e o limite que produz precisao.

## 2. Repetir tambem constroi

Uma pratica so ganha espessura quando deixa de depender de entusiasmo inicial. Repetir e menos excitante que comecar, mas costuma ser muito mais decisivo.

## 3. O arquivo importa

Guardar o que foi feito muda a relacao com o tempo. Um conjunto de textos, mesmo pequeno, ja funciona como mapa: mostra recorrencias, desvios e obsessoes.

Talvez este site exista exatamente para isso. Nao para anunciar uma conclusao, mas para acumular vestigios legiveis.
